Ce projet est inspiré du jeu {\bfseries{Laying Grass}} tiré de l\textquotesingle{}émission {\itshape The Devil\textquotesingle{}s Plan} sur Netflix (épisode 8). Le but du jeu est d’étendre son territoire en plaçant des tuiles d\textquotesingle{}herbe sur une grille pour créer la plus grande zone carrée possible.\hypertarget{md__c_1_2_users_2clemd_2_c_lion_projects_2_laying-_game_2_r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Règles du Jeu}{Règles du Jeu}}\label{md__c_1_2_users_2clemd_2_c_lion_projects_2_laying-_game_2_r_e_a_d_m_e_autotoc_md1}

\begin{DoxyEnumerate}
\item {\bfseries{Nombre de joueurs}} \+: 2 à 9 joueurs
\begin{DoxyItemize}
\item Grille 20x20 pour 2-\/4 joueurs
\item Grille 30x30 pour 5-\/9 joueurs
\end{DoxyItemize}
\item {\bfseries{But du Jeu}} \+:
\begin{DoxyItemize}
\item Le joueur ayant la plus grande zone carrée d’herbe à la fin des 9 tours remporte la partie.
\item En cas d’égalité, le joueur ayant le plus grand nombre de cases couvertes gagne.
\end{DoxyItemize}
\item {\bfseries{Déroulement de la partie}} \+:
\begin{DoxyItemize}
\item Chaque joueur reçoit une tuile de départ 1x1.
\item Chaque joueur commence avec un coupon d’échange de tuile.
\item Au début de chaque tour, le joueur reçoit une tuile d\textquotesingle{}herbe aléatoire parmi 96 formes possibles.
\end{DoxyItemize}
\item {\bfseries{Placement des Tuiles}} \+:
\begin{DoxyItemize}
\item Les tuiles doivent être placées adjacentes aux autres tuiles du joueur.
\item Elles ne doivent pas chevaucher les tuiles des autres joueurs.
\item Les tuiles peuvent être tournées ou retournées.
\end{DoxyItemize}
\item {\bfseries{Bonus}} \+:
\begin{DoxyItemize}
\item {\bfseries{Échange de tuile}} \+: Donne un coupon pour échanger de tuile.
\item {\bfseries{Pierre}} \+: Bloque une case et empêche d’y placer d’autres tuiles.
\item {\bfseries{Vol de tuile}} \+: Permet de voler une tuile d’un autre joueur.
\end{DoxyItemize}

Le jeu se joue dans le terminal. À chaque tour, les joueurs peuvent \+:
\begin{DoxyItemize}
\item Placer leur tuile selon les règles de proximité avec leur territoire.
\item Utiliser des commandes pour pivoter ou retourner les tuiles.
\item Utiliser un coupon pour échanger une tuile.
\end{DoxyItemize}

Les informations de jeu (tuiles disponibles, grille de jeu, etc.) sont affichées en ligne de commande.
