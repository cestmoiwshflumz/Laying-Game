\chapter{Documentation du projet Laying Game}
\hypertarget{index}{}\label{index}\index{Documentation du projet Laying Game@{Documentation du projet Laying Game}}
\hypertarget{index_intro_section}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_section}
Ce projet est un jeu stratégique où les joueurs placent des tuiles sur un plateau. Chaque joueur commence à partir d\textquotesingle{}un point de départ, et le but est de former le plus grand carré possible ou d\textquotesingle{}occuper un maximum de cases.\hypertarget{index_rules_section}{}\doxysection{\texorpdfstring{Règles du jeu}{Règles du jeu}}\label{index_rules_section}

\begin{DoxyItemize}
\item Chaque joueur a 9 tours pour jouer.
\item Les tuiles doivent être placées adjacentes à leurs propres cases ou à leur point de départ.
\item Les points sont calculés en fonction des cases occupées et de la taille du plus grand carré.
\end{DoxyItemize}\hypertarget{index_structure_section}{}\doxysection{\texorpdfstring{Structure du projet}{Structure du projet}}\label{index_structure_section}

\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_board}{Board}}\+: Gère le plateau de jeu.
\item {\ttfamily \doxylink{class_game}{Game}}\+: Contrôle la logique du jeu.
\item {\ttfamily \doxylink{class_player}{Player}}\+: Représente un joueur.
\item {\ttfamily \doxylink{class_tiles}{Tiles}}\+: Gère les tuiles disponibles.
\end{DoxyItemize}\hypertarget{index_usage_section}{}\doxysection{\texorpdfstring{Utilisation}{Utilisation}}\label{index_usage_section}

\begin{DoxyEnumerate}
\item Lancez le programme.
\item Entrez vos actions lors de chaque tour.
\item Consultez les scores finaux après 9 manches. 
\end{DoxyEnumerate}
